## Installation

```bash
$ npm install
```

```bash
#Start the database container
$ docker-compose up -d
```

## Running the backend server (Nest.js)

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running the frontend server (Vue.js)

```bash
# development
$ npm run serve --watch
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
